# condo-manager

Project to manage building receipts and expenses.

Technologies used:
- Java 21
- Quarkus
- [HTMX](https://htmx.org/)
- TailwindCSS
- [DaisyUI](https://daisyui.com/)
- ~~PlanetScale~~
- [Turso](https://turso.tech/) Websocket implementation

Features:
- CRUD (buildings, apartments, receipts, users)
- Gmail integration
- PDF generation
- Telegram notification
- Cron jobs
- Excel parsing
